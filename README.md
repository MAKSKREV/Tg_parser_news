# Tg_parser_news

Telegram-агент для парсинга новостей, обработки текста через AI и стилизации изображений.

## Описание

Бот выполняет следующие задачи:
1. **Парсинг новостей** из Telegram канала (ID: -1003808796392)
2. **Обработка текста** через OpenRouter API (модель deepseek/deepseek-v4-flash:free) - переписывает новость в ярком, кликбейтном стиле с хештегами
3. **Стилизация изображений** через OpenRouter API - применяет киберпанк/неон эффект к картинкам
4. **Отправка результата** в целевой чат (ID: -1003721535372)

## Требования

- Python 3.8+
- Библиотеки: `aiogram`, `aiohttp`, `logging`

## Установка

```bash
pip install aiogram aiohttp
```

## Запуск

```bash
python bot.py
```
